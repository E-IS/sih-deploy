Data collecting

Allegro-ObsDeb is used to input observation data, collecting on all Seychelles landing sites. All data are centralized into a central database (Harmonie).

Allegro-ObsDeb is a desktop software, that need to be downloaded and uncompressed before using on desktop or laptop computers:

Download Allegro-ObsDeb (Windows 32bits)
